# Aegis Nexus

An autonomous multi-agent strategic sandbox simulating emergent factions, diplomacy, conflict, trade, and AI-driven reasoning.

Live app: https://ai.studio/apps/0854d2e6-8626-48cd-b3fe-b85dda14f96c

## Run locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env` and set your `GEMINI_API_KEY` and `APP_URL`.
3. Run the app:
   `npm run dev`

## Build and run

1. Build the app:
   `npm run build`
2. Start the server:
   `npm run start`
